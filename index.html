What is Async Twilight Imperium?

Async Twilight Imperium is a Discord-based adaptation of Twilight Imperium 4th Edition that allows players to enjoy the game without the need for everyone to be online simultaneously. Players take turns through Discord commands while using a web interface to view the game state.

Key Features

  • Asynchronous Play - Take turns when convenient
  • Discord Integration - Full bot support with button control and slash commands
  • Web Interface - Real-time game board viewer
  • Complete Rules - Full Twilight Imperium 4E implementation including POK and Codex I-III
  • Community Support - Active Discord community
